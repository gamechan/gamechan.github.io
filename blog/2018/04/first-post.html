概要

Pelican導入から公開までのメモ

環境

• linux mint 18.3
• Python 3.6.3
• pip 9.0.3

導入

インストール

$ sudo pip install pelican Markdown pytz ghp-import

クイックスタート

$ git clone https://github.com/gamechan/gamechan.github.io  
$ cd gamechan.github.io  
$ pelican-quickstart

記事の作成

  • markdownで記事を作成して/contentに保存
  • テンプレ
Title: 記事タイトル  
date: yyyy-mm-dd HH:mm  
modified: yyyy-mm-dd HH:mm  
tags: tag1, tag2  
category: category名  
slug: my-super-post  
authors: 投稿者  
summary: Short version for index and feeds

pelicanconf.pyの設定変更

変更
SITEURL = '' → SITEURL = 'http://gamechan.github.io'
#RELATIVE_URLS = True → RELATIVE_URLS = True

追加 

TIMEZONE = 'Asia/Tokyo'  
DATE_FORMATS = {  
    'en': '%a, %d %b %Y',  
    'ja': '%Y-%m-%d(%a)',  
}  
ARTICLE_URL = 'blog/{date:%Y}/{date:%m}/{slug}.html'  
ARTICLE_SAVE_AS = 'blog/{date:%Y}/{date:%m}/{slug}.html'  
PAGE_URL = 'pages/{slug}.html'  
PAGE_SAVE_AS = 'pages/{slug}.html'  
YEAR_ARCHIVE_SAVE_AS = 'blog/{date:%Y}/index.html'  
MONTH_ARCHIVE_SAVE_AS = 'blog/{date:%Y}/{date:%m}/index.html'  
CATEGORY_URL = 'blog/category/{slug}/'  
CATEGORY_SAVE_AS = 'blog/category/{slug}/index.html'  
CATEGORIES_SAVE_AS = 'blog/categories.html'  
TAG_URL = 'blog/tag/{slug}/'  
TAG_SAVE_AS = 'blog/tag/{slug}/index.html'  
TAGS_SAVE_AS = 'blog/tags.html'  
AUTHOR_URL = 'author/{slug}.html'  
AUTHOR_SAVE_AS = 'author/{slug}.html'  
AUTHORS_SAVE_AS = 'authors.html'

ローカルで確認

$ make html  
$ make serve

http://localhost:8000にアクセス

記事の公開

$ make github  
$ ghp-import outputh  
$ git push https://github.com/gamechan/gamechan.github.io
